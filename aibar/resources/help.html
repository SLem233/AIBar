AIBar

Дашборд лимитов AI-провайдеров для Windows — аналог CodexBar-KDE

Что показывают кольца

  • Внешнее кольцо — текущая сессия (обычно окно 5 часов); её процент — в центре.
  • Внутреннее кольцо — недельный (или месячный) лимит.

Иконка в трее показывает самого «загруженного» провайдера.

Виджет на рабочем столе

  • Перетаскивание — зажмите левую кнопку в любом месте виджета.
  • Размер — потяните за уголок справа внизу. Позиция и размер запоминаются.
  • Детали — наведите мышь: рядом всплывёт панель с тарифом, всеми окнами лимитов и временем до сброса (значок ↺).
  • Мини-режим (правая кнопка → «Мини-режим») — на виджете остаются только провайдеры с расходом лимита 70%+; если таких нет, показывается самый активный за последние 15 минут. Ховер-панель и дашборд всегда показывают всех.
  • ⏸ нет VPN — Claude, Codex и OpenAI API недоступны без VPN (гео-блок). Когда VPN выключен, запросы к ним (включая токены) не отправляются: на виджете появляется жёлтая плашка «⏸ нет VPN», у тайлов — значок ⏸, кольца держат последние значения. При включении VPN всё оживает само на ближайшем цикле.
  • Меню — правая кнопка: обновить, мини-режим, настройки, справка, скрыть, выход.
  • Включить/выключить виджет: меню иконки в трее → «Виджет поверх окон».

Провайдеры и авторизация

ПровайдерЧто нужно
ClaudeНичего — используется токен Claude Code (~/.claude). Если истёк, запустите claude.
CodexНичего — токен codex CLI (~/.codex). Если истёк, запустите codex.
CursorНичего — сессия установленного приложения Cursor.
Z.ai (zcode)API-ключ coding-плана: z.ai → Manage API Key → Coding Plan. Вставьте в Настройках (регион Global или BigModel CN). Подхватывается и из переменной Z_AI_API_KEY.
OpenCodeCookie auth с opencode.ai: DevTools (F12) → Application → Cookies → скопируйте строку auth=… в Настройки. Workspace определяется сам.
OpenAI APIAdmin-ключ (platform.openai.com → Settings → Organization → Admin keys). Обычный ключ проекта не подойдёт. Показывает расход за месяц и остаток на счету: введите в Настройках остаток со страницы биллинга и дату — дальше расходы вычитаются автоматически. Бюджет $/мес даёт кольцо с процентом.
Google (Gemini)Вход в gemini CLI (токен из ~/.gemini). Показывает квоты моделей Pro/Flash. Баланс AI Studio API Google программно не отдаёт.
TavilyОбычный API-ключ tvly-… — квота кредитов плана за месяц. Подхватывается и из TAVILY_API_KEY.

Serper.dev не даёт API для проверки баланса — его мониторить нельзя (только вручную в их дашборде).

AIBar только читает локальные токены и ничего не отправляет никуда, кроме официальных API провайдеров.

Настройки

Правая кнопка по виджету или иконке в трее → «Настройки…»: включение провайдеров, ключи Z.ai/OpenCode, интервал опроса (1–60 мин). Файл конфигурации: %APPDATA%\AIBar\config.json.

Автозапуск и запуск

  • Запуск: AIBar.exe — двойной клик, ничего устанавливать не нужно.
  • Автозапуск: Win+R → shell:startup → положите туда ярлык на AIBar.exe.
  • Обновления: при выходе новой версии на виджете появляется плашка «↓ Update available» — клик открывает страницу загрузки.
  • Первый запуск чуть медленнее (распаковка), в диспетчере задач два процесса AIBar — это норма.

Если что-то не так

  • ⚠ в карточке — текст ошибки там же: чаще всего истёкший токен (перезапустите соответствующий CLI) или нет сети.
  • 401 у Claude/Codex — выполните claude / codex в терминале, токен обновится сам.
  • ⏸ нет VPN — гео-блок: включите VPN, данные обновятся сами на ближайшем цикле. Пока VPN выключен, приложение не отправляет запросы (и токены) к Claude, Codex и OpenAI API.
  • Пусто у Z.ai — проверьте регион (Global ↔ BigModel CN) и ключ.
